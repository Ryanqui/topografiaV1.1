Practicas de Topografia
======================
Autor visible en la interfaz: Jesus Alfonso Barrante Flores

Aplicativo web responsive para escritorio y celular, orientado al registro, cálculo y visualización de prácticas de topografía y taquimetría.

Ajustes aplicados en esta versión:
- Cambio de marca visible: “Practicas de Topografia”.
- Cambio de subtítulo visible: “Jesus Alfonso Barrante Flores”.
- Eliminado el texto “Modo académico” de la interfaz.
- Mejorado el estilo visual de los gráficos.
- Corregida la visualización de textos en secciones transversales para evitar sobreposición.
- Mejorado el perfil longitudinal tipo guitarra.
- Añadidas más bandas al perfil tipo guitarra: terreno natural, rasante de proyecto, tubería, cota roja, pendiente y observaciones.
- Añadidos botones para exportar gráficos como PNG.
- Añadida opción de exportación de gráficos a PDF mediante ventana de impresión.

Archivos principales:
- index.html: interfaz principal.
- styles.css: diseño visual responsive.
- app.js: lógica del sistema, cálculos, tablas editables y gráficos.
- README.txt: guía general.

Uso rápido:
1. Abra index.html en un navegador moderno.
2. Cree o seleccione un proyecto.
3. Complete los datos generales.
4. Ingrese a cada módulo y registre los datos.
5. Use los botones de agregar, insertar, duplicar, borrar, importar y exportar.
6. Use los botones de “Exportar PNG” o “Exportar PDF” en cada gráfico.
7. Use “Guardar” y exporte respaldo JSON para seguridad.
